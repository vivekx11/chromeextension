# Chrome Extension.

A powerful and lightweight Chrome extension that enhances your browsing experience.

## Features

- 🚀 Lightweight and fast
- 📦 Easy to install and use
- 🎨 Beautiful UI with modern design
- 🔧 Easy to customize
- 📱 Responsive design
- 🌙 Dark/Light mode support

## Installation

1. Clone this repository
   ```bash
   git clone https://github.com/vivekx11/chromeextension.git
   ```

2. Open Chrome and go to `chrome://extensions/`

3. Enable "Developer mode" (toggle in top right)

4. Click "Load unpacked" and select the cloned folder

5. The extension will appear in your extensions list

## Usage

- Click the extension icon in the Chrome toolbar
- Follow the on-screen instructions
- Customize settings as needed

## Dependencies

No external dependencies are required. This extension uses vanilla JavaScript and built-in Chrome APIs.

For development tools, see `requirements.txt`

## Development

### Prerequisites

- Chrome browser (v90+)
- Code editor (VS Code recommended)

### Setup

1. Clone the repository
2. Open the folder in your code editor
3. Make changes to the files
4. Reload the extension in Chrome (`chrome://extensions/` → Click refresh)

## File Descriptions

- **manifest.json**: Defines the extension's permissions, metadata, and scripts
- **popup.html**: The HTML structure of the extension popup
- **popup.js**: JavaScript logic for the popup functionality
- **style.css**: CSS styling for the extension interface
